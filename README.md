# android-homework

本仓库用于提交 Android 课程测试作业，当前项目基于开源记账本应用 Mine 进行测试实践。

## 原项目简介

Mine 是一个记账本应用，界面简单干净。

应用中使用的图标来自 [iconfont](http://iconfont.cn/) 和 [material-design-icons](https://github.com/google/material-design-icons)。

[应用宝下载](http://sj.qq.com/myapp/detail.htm?apkName=com.coderpage.mine)

## 当前进度

- 已整理 Android 项目源码。
- 已添加单元测试、Robolectric 测试与 Espresso 测试相关内容。
- 已添加 Jenkins 持续集成脚本和初步报告。
